# 济宁楼盘数据抓取与分析工具

这是一个用于抓取济宁市商品房房源信息并生成销售报告的工具。它能够自动获取最新的房源状态（可售、已售、签约中、抵押等），并生成 Markdown、PDF 和 JSON 格式的多维度报告。

## 🚀 主要功能

- **自动化抓取**：基于小区代码（ProjectID）自动抓取所有楼栋及房源明细。
- **状态分析**：智能识别房源的最新销售状态。
- **多格式输出**：
  - **MD / PDF**：可直接查看的图文报告。
  - **JSON**：结构化数据，方便程序二次处理。
- **对比分析**：提供专门的脚本对比不同日期的销售数据，精准掌握每日网签变化。

## 📦 环境准备

1. **Node.js**: 确保已安装 Node.js (推荐 v16+)。
2. **包管理器**: 推荐使用 `pnpm`。
3. **依赖安装**:
   ```bash
   pnpm install
   ```

## 🛠 使用方法

### 1. 查询小区数据
运行 `query_project.js` 并传入小区代码。
```bash
node query_project.js <小区代码>
# 示例：查询星都会
node query_project.js 00000129
```
*提示：输出文件将保存在 `output/小区名_小区代码/` 目录下，文件名格式为 `小区名_YYYYMMDD.ext`。如果在同一天多次运行，旧文件将被自动覆盖。*

### 2. 批量查询
使用脚本批量抓取预设的重点小区：
```bash
./fetch_selected.sh
```

### 3. 数据对比分析
对比同一小区的最新两份数据，查看每日增量变化：
```bash
# 对比所有预设小区
node compare_today.js

# 或者查看特定小区的分析汇总
node analyze_all.js
```

## 📂 目录结构

- `output/`: 存放所有抓取的结果和报告。
- `projects_data.json`: 预存的小区代码参考列表。
- `query_project.js`: 核心抓取脚本。
- `compare_today.js`: 每日网签增量分析。
- `fetch_selected.sh`: 批量运行脚本。

## ⚠️ 注意事项

- **PDF 生成**: PDF 生成依赖于 Puppeteer (md-to-pdf)，首次运行可能需要下载 Chromium 浏览器。
- **接口延迟**: 为避免对目标网站造成压力，脚本内置了请求延迟。
- **小区代码**: 小区代码可以在济宁政务服务网查询获取，或参考 `projects_data.json`。
